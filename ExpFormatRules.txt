Key: unformatted -> formatted

1. All variables must have an exponent, surrounded by parentheses
    3x^2 -> 3x^(2)
    5x -> 5x^(1)

2. All variables must have an explicit coefficient
    x -> 1x^(1)

3. Distribution into a parenthesized expression must be explicit, using a '*'
    3(x + 4) -> 3 * (x + 4)
